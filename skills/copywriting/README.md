# Copywriting Assistant Skill

Professional copywriting for ads, emails, landing pages, and marketing materials. Specializes in conversion-focused, persuasive writing with psychological triggers and emotional appeal.

## Features

- **Ad Copy**: Write compelling ads for Google, Facebook, LinkedIn, Instagram, Twitter
- **Email Marketing**: Create subject lines and campaign bodies
- **Landing Pages**: Write persuasive pages optimized for conversions
- **Social Media**: Create engaging platform-specific posts
- **Psychological Triggers**: Use FOMO, social proof, urgency, scarcity
- **A/B Testing**: Generate copy variations
- **Benefit-Focused**: Emphasize transformation and customer benefits

## Tools

### write_ad_copy
Generate ads optimized for specific platforms.

```
Agent: "Write a Facebook ad for our project management tool"
Platform: facebook
Product: Project management software
TargetAudience: Remote teams and freelancers
MainBenefit: Save 5 hours per week on project management
CTA: Start free trial
```

### write_email_campaign
Create complete email marketing campaigns.

```
Agent: "Write a promotional email for our summer sale"
CampaignType: promotional
Subject: 30% off summer sale
Audience: Existing customers
KeyMessage: Limited-time discount on all products
```

### write_landing_page
Design persuasive landing page copy.

```
Agent: "Write landing page copy for our new course"
ProductName: Advanced Copywriting Course
TargetAudience: Aspiring copywriters and marketers
MainProblem: High cost of copywriting courses and poor quality
UniqueValue: AI-powered feedback and real client projects
DesiredAction: Enroll in course
```

### write_social_media
Create platform-optimized social posts.

```
Agent: "Write a LinkedIn post about productivity tips"
Platform: linkedin
Topic: Time management for entrepreneurs
Tone: professional
Engagement: comments and shares
```

## Configuration

- `tonePref`: Your brand's voice (professional, casual, friendly, bold, humorous)
- `audienceLevel`: Reading complexity (beginner, intermediate, advanced) - default: intermediate
- `focusOnBenefits`: Emphasize customer benefits over features (default: true)
- `usePowerWords`: Include emotional power words (default: true)

## Copywriting Elements

### Headlines
- Attention-grabbing
- Benefit-focused
- Curiosity-driven
- Question-based

### Body Copy
- Problem-agitate-solve format
- Social proof and authority
- Objection handling
- Clear value proposition

### Calls-to-Action
- Action-oriented verbs
- Urgency or scarcity
- Low-friction next steps
- Multiple CTA variations

## Platform Specifications

Each platform receives optimized copy considering:
- Character limits
- Platform best practices
- Algorithm preferences
- User behavior
- Engagement drivers

## Output

Copywriting deliverables include:
- Multiple variations (A/B test options)
- Tailored to platform and audience
- With strategic reasoning
- Ready to implement
- Performance optimization tips
