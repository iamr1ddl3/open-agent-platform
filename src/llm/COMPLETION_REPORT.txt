================================================================================
API REFERENCE
================================================================================

Gateway Methods:
  - configureProvider(provider, config): Configure a provider
  - configureMultiple(configs): Configure multiple providers
  - setDefaultProvider(provider): Set default provider
  - getDefaultProvider(): Get current default
  - listProviders(): List all available providers
  - getProvider(provider): Get provider instance
  - complete(request, provider?): Get completion
  - stream(request, provider?): Stream completion
  - testConnection(provider?): Test provider connection
  - listModels(provider?): Get available models
  - testAllConnections(): Test all providers

Type Definitions:
  - LLMProvider: Provider type union
  - LLMProviderConfig: Configuration interface
  - LLMMessage: Message with role and content
  - LLMRequest: Completion request
  - LLMResponse: Completion response
  - LLMStreamChunk: Streaming chunk
  - ToolCall: Tool invocation
  - ToolDefinition: Tool definition
  - LLMProviderInterface: Provider contract

================================================================================
USAGE EXAMPLE
================================================================================

import { getLLMGateway } from './src/llm';

const gateway = getLLMGateway();

gateway.configureProvider('anthropic', {
  apiKey: process.env.ANTHROPIC_API_KEY,
  model: 'claude-3-5-sonnet-20241022',
});

const response = await gateway.complete({
  messages: [
    { role: 'system', content: 'You are helpful' },
    { role: 'user', content: 'Hello!' }
  ],
  maxTokens: 1000,
  temperature: 0.7,
});

console.log(response.content);
console.log('Tokens:', response.usage);
